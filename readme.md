# HOW TO 

Install dependencies :
```sh
pip install > requirements.txt
```

Run app in web :
```sh
streamlit run app.py
```

### WHAT CAN YOU DO WITH THIS APP
1. Test a string on DFA
2. Test a string on Regex that generates NFA
3. Minimize DFA
4. Check if 2 DFAs are equivalent
